node.js가 설치되어 있어야합니다.

현재 추가된 React는 port 3000번에서 실행되며 SpringBoot는 port 8080에서 실행되게 됩니다. 개발시 port 3000에서 8080으로 넘겨주는 역활을 할 예정입니다.

9. 배포용 패키지 생성
